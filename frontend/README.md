# Hoster Frontend

Frontend application for the Hoster company website.

This frontend provides the public landing pages and a simple admin
interface, and communicates with the backend via REST APIs.

---

## Tech Stack
- React (Vite)
- Tailwind CSS
- React Router
- Fetch API

---

## Features
- Company landing pages
- Contact form
- Careers page
- Admin dashboard
- Backend API integration

---

## Run Locally

```bash
npm install
npm run dev
